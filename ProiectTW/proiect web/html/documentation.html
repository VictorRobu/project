Introduction

Auto Park Smart Explorer is a web application that provides information about car models, manufacturers, and more from every county in Romania. This documentation aims to guide users through the process of using the application and understanding its features.

Getting Started

To get started with Auto Park Smart Explorer, users need to sign up for an account or log in if they already have an account. After logging in, users can access the main features of the application, such as searching for car data and viewing different types of visualizations.

Features

Auto Park Smart Explorer offers several features to help users explore and analyze car data. These features include:

  • Search and filter functionality to find specific car data.
  • Various visualization options, such as bar charts, pie charts, and area charts.
  • Customizable settings, including theme and font selection.
